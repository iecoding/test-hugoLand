> ¡Es la mayor versión desde la v5 misma—**Bootstrap v5.2.0-beta1 está aquí!** Esta versión incluye documentación rediseñada, variables CSS para todos nuestros componentes, offcanvas responsivo, nuevos ayudantes y utilidades, botones e inputs refinados, y muchas mejoras bajo el capó.

Dado el tamaño de la actualización y el tiempo desde nuestra última versión, estamos haciendo algo diferente y lanzándola primero como beta. [Sigue leyendo para más detalles](https://blog.getbootstrap.com/2022/05/13/bootstrap-5-2-0-beta/).
