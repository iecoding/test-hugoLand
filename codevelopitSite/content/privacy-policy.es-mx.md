# Política de Privacidad

**TL;DR**: No usamos cookies y no recopilamos ningún dato personal.

## Visitantes del sitio web

- No se recopila información personal.
- No se almacena información en el navegador.
- No se comparte, envía o vende información a terceros.
- No se comparte información con empresas publicitarias.
- No se extrae ni se recopila información para tendencias personales y de comportamiento.
- No se monetiza la información.

## Información que recopilamos y para qué la usamos

El servidor que aloja este sitio puede recopilar la siguiente información:

- URL de la página
- Referente HTTP
- Navegador
- Sistema operativo
- Tipo de dispositivo
- Dirección IP

Todos los registros se descartan en cada versión.
