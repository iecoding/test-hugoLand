> ¡Es oficial, la versión estable final de v5.3.0 ha llegado! Ha sido un esfuerzo monumental renovar nuestra base de código para variables CSS y modos de color, uno que verá cambios continuos que conducirán a un eventual Bootstrap 6. ¡Y estamos tan emocionados de finalmente lanzarla!

Además de todo el trabajo que se ha puesto en esta versión, ha pasado mucho detrás de escena desde que lanzamos nuestras pre-versiones. Sigue leyendo para aprender todo lo que es [nuevo en v5.3.0](https://blog.getbootstrap.com/2023/05/30/bootstrap-5-3-0/).
